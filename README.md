# CwithM ( Certificate WITH Major )

##  ✨description

> 자신의 전공을 입력하면 관련된 자격증을 보여주는 웹사이트
>
> 관심있는 자격증을 하트를 눌러서 따로 관리 할 수 있다.
>
> **프로젝트 기간 2020.11.04 ~ 2020.12.16**
